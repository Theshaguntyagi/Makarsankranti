# Makar Sankranti Animation

## Description

This project is a simple web animation created using HTML and CSS to celebrate the festival of Makar Sankranti. The animation features flying kites and a festive wish displayed on the screen. Additionally, there are animated clouds in the background to enhance the visual appeal.

## Getting Started

To view the animation, simply open the `index.html` file in a web browser. You can also deploy the project to a web server for online access.

## Clone the repository
git clone https://github.com/your-username/makar-sankranti-animation.git

## Change directory
cd makar-sankranti-animation

Open index.html in a web browser

## Usage

The animation will start automatically when you open the index.html file. The flying kites and clouds, along with the festive wish, create a visually appealing representation of Makar Sankranti.

## Customization

You can customize the project by modifying the style.css file. You can change the background color, adjust the animation speed, or replace the wish message and image. Additionally, you can experiment with different kite colors and sizes.
